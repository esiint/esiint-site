# ESIINT — Site institucional em Next.js

Projeto inicial pronto para deploy na Vercel.

## Como rodar localmente

```bash
npm install
npm run dev
```

## Estrutura principal

- `app/page.tsx` → homepage
- `app/observatorio/page.tsx` → página do Observatório
- `app/cursos/observatorios-centros-de-evidencia/page.tsx` → curso principal
- `app/cursos/a-era-da-inteligencia/page.tsx` → curso introdutório
- `app/projetos/prevencao-seguranca-riscos/page.tsx` → projeto de Prevenção 4.0

## Próximos passos recomendados

1. Inserir imagens e artes oficiais.
2. Conectar formulário real de contato.
3. Adicionar favicon e identidade visual final.
4. Fazer deploy na Vercel.
